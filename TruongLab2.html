Terms and Definitions

How to define a table?
A table in HTML consists of table cells inside rows and columns.
Describe when to use the following HTML elements:
  • tr: Use to make a new table row.
  • th: Use to have table header cells.
  • td: Defines a cell in a table.
  • thead: Groups the header content in a table.
  • tfoot: Groups the footer content in a table.
  • caption: Defines the table caption.
What is the difference between colspan and rowspan?
Colspan combines columns and rowspan combines rows; both dependent on how many columns/rows you want to combine.
What is the difference between cells spacing and padding?
Cell padding is the space between the cell edges and the cell content whereas cell spacing is the space between each cell.
How do you define table borders using CSS? How do you control border widths using CSS?
Table borders are defined by "table, th, td {border: 1px solid; }." You can control border widths by changing the px size.
