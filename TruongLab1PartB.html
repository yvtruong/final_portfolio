What is DOCTYPE statement?
A DOCTYPE statement is information to the browser about what document type to expect.
How do you use: < ol >;< ul >;< dl >
If you want to make an ordered list, start with the < ol > tag and make each list start with the < li > tag. The same will be done with an unordered list except the < ol > tag will be < ul >. The < dl > tag defines a description list and it is used in conjuction with < dt > (defines terms/names) and < dd > (describes each term/name)
Difference between text editor and word processor.
Text editor provides the ability the add, change, and remove characters from a file. Word processor has a number of options where you can format the text or document.
What is the client-server model?
The client server model is a distributed application structure that partitions task or workload between the providers of a resource or service, called servers, and service requesters called clients.
Provide a description for the following network protocols:
  • FTP: The "File Transfer Protocol" refers to a group of rules that govern how computers transfer files from one system to another over the internet.
  • HTTP: The "Hypertext Transfer Protocol" is a networking protocol for distributed, collaborative, hypermedia information systems.
  • TCP/IP: The "Transmission Control Protocol/Internet Protocol is a set of standardized rules that allow computers to communicate on a network such as the internet.
